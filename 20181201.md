# Indirect

Google Spreadsheet 등의 스프레드시트에서,  
다른 시트의 값을 불러올때는 아래 수식을 사용한다.

```
='시트이름'!A1
```

`시트이름`에 해당하는 값을 변수처럼 사용하고 싶을 경우 `INDIRECT` 함수를 사용한다.

C3 셀의 값이 `시트이름` 이라고 하면,

```
=indirect(C3&"!A1")
```

로 불러 올 수 있다.

---

## 시트이름과 불러올 값을 변수로 사용할 경우

C1 셀 값이 "예산시트" 이고, C2 셀 값이 F10 이면,

```
=indirect(C1&"!"&C2)
```

위 수식은,

```
='예산시트!C2'
```

와 같다.
